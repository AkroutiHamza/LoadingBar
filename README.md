
# LoadingBar.sh

> Progress bar in bash by Adriano Pinaffo.
This progressbar will fill all the columns of the terminal window.

### Preview
![LoadingBar.sh in action on terminal](./LoadingBar.gif)

### Usage:
        ./LoadingBar.sh [SECONDS] 

### Example for 10 seconds:
        ./LoadingBar.sh 10
